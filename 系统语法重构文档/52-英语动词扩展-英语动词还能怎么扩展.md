# 英语动词扩展：英语动词还能怎么扩展？

前面几大块已经把最基础的主线搭起来了：

- 句子结构
- 句型变化
- 时态和语态
- 名词与限定词
- 修饰成分

但到了这里，很多人会发现还有一批高频结构总是反复碰到：

- can / should / must
- have to
- to do / doing / done
- should have done
- must have done

这些内容都和动词有关，但它们已经不是最基础的“主语 + 动词 + 时态”那一层了。

这篇文章只解决一个总问题：

> **英语动词在基础时态之外，还会怎样继续扩展。**

---

## 一、这个模块到底在解决什么

这个模块主要解决三类问题：

### 1. 动作不是只分“现在、过去、将来”

有时候你还想表达：

- 能不能
- 该不该
- 必不必须
- 可不可能

这时就会用到情态动词。

### 2. 一个句子里不只会出现一个动词形式

例如：

- I want to go.
- Reading helps me relax.
- The broken window was fixed.

这里出现了：

- to do
- doing
- done

这就进入了非谓语动词。

### 3. 你不只想说动作本身，还想对动作做判断

例如：

- You should have told me.
- He must have been tired.

这类结构会把“情态”和“已经发生的动作”连起来。

---

## 二、这一模块可以先分成两大主线

### 1. 情态动词主线

这一条主要在解决：

- 能力
- 许可
- 义务
- 建议
- 推测

高频结构包括：

- can
- could
- must
- should
- have to

再往后会进入：

- should have done
- could have done
- would have done
- must have done

### 2. 非谓语动词主线

这一条主要在解决：

- 一个句子里为什么会有多个“像动词的东西”
- to do、doing、done 分别在干什么
- 为什么有时要接动名词，有时要接不定式
- 分词为什么能拿来修饰名词或整句

---

## 三、为什么很多人会觉得这一块难

因为这一模块的特点不是“新单词多”，而是：

> **同一个动词，会以不同形式出现在不同位置，表达不同层次的意思。**

比如：

- I can swim.
- I want to swim.
- Swimming is fun.
- The broken glass was dangerous.

四句都跟动词有关，但逻辑完全不一样。

所以这块最怕的不是不背，而是没有先分清：

- 这是在表达态度判断
- 还是在扩展动词结构

---

## 四、先记住这一模块的学习顺序

最自然的顺序是：

1. 先掌握最常见的情态动词
2. 再看几组最容易混的情态结构
3. 然后进入非谓语动词的基本地图
4. 再拆 to do、doing、done 的细分用法
5. 最后再处理情态完成式这种更进阶的结构

所以后面会按这个顺序展开：

- 情态动词怎么理解
- must 和 have to 怎么区分
- can、could、be able to 怎么区分
- 非谓语动词到底是什么
- 动名词和不定式怎么区分
- 分词作定语怎么用
- 分词作状语怎么用
- should have done、could have done、would have done 怎么区分
- must have done、may have done、might have done、can't have done 怎么区分

---

## 五、先建立一张最简地图

你可以先这样粗略记：

- `can swim`：在说能力
- `must leave`：在说义务
- `want to leave`：在说想去做
- `reading books`：在说“读书这件事”
- `a broken cup`：在说“被打破的杯子”
- `should have called`：在说“本来应该已经打电话”

也就是说：

- 情态动词让你对动作加态度
- 非谓语结构让你把动作变成句子里别的成分

---

## 六、先做一个小判断

看下面这些加粗部分，各自更接近哪一类：

1. She **can speak** French.
2. I hope **to see** you soon.
3. **Swimming** is good exercise.
4. The **broken** chair is mine.
5. You **should have told** me earlier.

---

## 七、参考答案

1. 情态动词表达能力
2. 不定式结构
3. 动名词结构
4. 分词作定语
5. 情态完成式

---

## 八、一句话总结

这一模块学的是：**动词除了表示动作和时态，还能继续表达态度、义务、推测，并且能变成句子里的其他成分。**
