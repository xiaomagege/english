# 英语语法里 each、every、all 怎么区分？都像“都、每个”，到底差在哪

很多英语学习者看到 each、every、all 时，会觉得它们好像都能表达：
- 每个
- 都
- 全部

但真正写句子时，很容易犹豫：
- each students ❌
- every students ❌
- all student ❌
- they each is ready ❌

这类问题非常常见。

这篇文章只讲一个重点：**each、every、all 到底怎么区分。**

---

## 一、先记住最核心的主线

如果你今天只先记一套最实用的区别，那就是：

> **each 强调一个一个地看，every 强调整体中的每一个，all 强调整体全部。**

例如：
- Each student has a book.
- Every student needs practice.
- All students are here.

先记住这个感觉，后面很多细节都会顺很多。

---

## 二、each 怎么用

each 最常见的是：

> **一个一个地看待个体。**

它通常后接：
- 单数可数名词

例如：
- each student
- each child
- each book

例句：
- Each student has a desk.
- Each child got a gift.

这里的重点是：
- 虽然说的是一群人
- 但视角是一个一个地看

---

## 三、every 怎么用

every 最常见的是：

> **在一个整体里，每一个都包括进去。**

它也通常后接：
- 单数可数名词

例如：
- every student
- every day
- every room

例句：
- Every student must finish the work.
- I go running every morning.

所以 each 和 every 都接单数，但感觉不完全一样。

---

## 四、all 怎么用

all 最常见的是：

> **全部，所有。**

它可以接：
- 复数可数名词
- 不可数名词

例如：
- all students
- all books
- all the water
- all the money

例句：
- All students passed the test.
- All the food was delicious.

所以 all 的关注点不是“一个一个”，而是“整体全部”。

---

## 五、each 和 every 有什么区别

这组很容易混。

### each
更强调个体感。
- Each student received a different task.

### every
更强调整体覆盖。
- Every student in the class was ready.

你可以先简单记：
- each = 一个一个看
- every = 整体里每个都算上

---

## 六、all 和前面两者有什么区别

- each / every 后面通常接单数名词
- all 后面通常接复数名词或不可数名词

例如：
- each book ✅
- every book ✅
- all books ✅
- all water ✅

但：
- each books ❌
- every books ❌

这点特别重要。

---

## 七、最实用的判断方法

你先问自己两个问题：

### 1. 你是在强调“一个一个”吗？
- 是 → each

### 2. 你是在强调“整体里所有都包括进去”吗？
- 强调整体中的每一个 → every
- 强调整体全部 → all

这个判断对初学阶段非常够用。

---

## 八、中国学习者最常见的错误

### 错误 1：each / every 后面接复数
- each students ❌
- each student ✅

- every books ❌
- every book ✅

### 错误 2：all 后面接单数可数名词
- all student ❌
- all students ✅

### 错误 3：把 each 和 all 完全当成一回事
- Each student has a name.
- All students have names.

两句都对，但关注点不同。

---

## 九、实用例句

- Each child has a different toy.
- Every room in the house is clean.
- All students are waiting outside.
- I read every page carefully.
- Each answer was different.
- All the money is gone.
- Every person needs rest.
- Each of us has a role.

---

## 十、做几个小练习

选择最合适的词填空。

### 1.
___ student has a notebook. （强调一个一个）

### 2.
___ students are ready. （全部）

### 3.
I drink coffee ___ morning. （每个早晨）

### 4.
___ answer was correct. （每一个）

### 5.
___ the food was fresh. （全部）

---

## 十一、参考答案

1. Each
2. All
3. every
4. Each / Every
5. All

---

## 十二、一句话总结

> **each 强调一个一个，every 强调整体中的每一个，all 强调整体全部。**

---

## 十三、结尾

只要你先分清“个体感”和“整体感”，这三个词就不会再总像一个意思。

因为它们常常一起出现，而一旦涉及“两者”，很多人就会开始混乱。
