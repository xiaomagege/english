# 英语语法系列博客规划

## 一、系列总目标

系列名称建议：**《英语语法通关指南》**

面向读者：
- 英语初学者
- 想系统补语法的人
- 备考用户
- 不喜欢传统语法书、但想把语法真正学明白的人

系列目标：
1. 按英语语法的核心系统来讲，而不是只做零散知识点堆叠
2. 先帮读者建立“句子怎么组成”的整体框架，再进入时态、句型、从句和易混点
3. 每篇只解决一个重点问题，但整套文章连起来要有清晰学习路径
4. 用最少术语讲清核心规则，保证读者能直接拿去造句和纠错

---

## 二、总体结构思路

这套博客后续应当以 **“先骨架，后变化；先简单句，后复杂句；先系统主干，后高频易错点”** 为总原则。

也就是说，整体顺序应该尽量遵循：

1. **先讲句子结构**
   - 句子由什么组成
   - 主语、谓语、宾语、表语这些核心成分怎么理解
   - 英语为什么不能随便换语序

2. **再讲基础时态系统**
   - 一般现在时、一般过去时、现在进行时、一般将来时、现在完成时
   - 以及这些高频时态之间怎么区分

3. **再讲句子的常见变形**
   - 否定句
   - 疑问句
   - there be
   - 祈使句
   - 情态动词

4. **再讲句子扩展能力**
   - 介词、频率副词、连词
   - 比较级、被动语态、非谓语
   - 从简单句走向更完整、更自然的表达

5. **最后讲复杂句和高频易错点**
   - 宾语从句、定语从句、状语从句
   - many / much / a lot of、some / any、few / little 等
   - say / tell / speak / talk、do / make 等高频混淆项

这样安排的好处是：
- 读者不会一上来就被零碎规则打散
- 每一篇都知道自己在整个系统里处在什么位置
- 文章既能单篇阅读，也能连起来形成课程感

## 三、推荐系列主结构

建议把整套内容固定分成 **7 个模块**，而不是只按“先写什么更容易起量”来组织。

### 模块 1：句子骨架与基础结构
这个模块解决“英语句子到底怎么搭起来”。

1. 英语语法到底在学什么
2. 英语句子到底怎么组成
3. 主语、谓语、宾语、表语怎么理解
4. be 动词到底怎么用
5. 英语疑问句怎么变
6. 英语否定句怎么变
7. there be 句型怎么用
8. 祈使句怎么用

核心目标：
- 先把句子的最小骨架讲清楚
- 让读者知道一个完整句子至少要抓什么
- 建立“主语 + 谓语 + 其他成分”的基本句感
- 先把主语、谓语、宾语、表语四个核心成分讲清，再进入 be 动词和句型变形

### 模块 2：基础时态系统
这个模块解决“动作发生在什么时候，状态怎么随时间变化”。

8. 一般现在时怎么用
9. 一般过去时怎么用
10. 现在进行时怎么用
11. 一般将来时怎么用
12. 现在完成时怎么用
13. 一般现在时和现在进行时怎么区分
14. 一般过去时和现在完成时怎么区分
15. 英语 12 时态到底怎么学
16. be going to 和 will 怎么区分
17. used to 和 be used to 怎么区分
18. used to、would、usually 怎么区分

核心目标：
- 让读者先掌握最高频时态
- 再通过对比文章建立真正的判断能力
- 最后用“12 时态怎么学”做系统收束，而不是一开始就压过去

### 模块 3：名词系统与限定词系统
这个模块解决“名词前后到底怎么搭，数量和指代怎么判断”。

19. a、an、the 怎么区分
20. this、that、these、those 怎么区分
21. other、another、the other、others 怎么区分
22. each、every、all 怎么区分
23. both、either、neither 怎么区分
24. many、much、a lot of 怎么区分
25. some 和 any 怎么区分
26. few、a few、little、a little 怎么区分

核心目标：
- 帮读者建立“可数 / 不可数”“单数 / 复数”“特指 / 泛指”的感觉
- 把数量、范围、指代、限定几个常见维度拆开讲

### 模块 4：修饰成分与补充信息
这个模块解决“怎样把句子说得更完整、更细、更自然”。

27. in、on、at 怎么用
28. 频率副词怎么用
29. very、too、so、quite 怎么区分
30. so 和 such 怎么区分
31. too、also、either 怎么区分
32. to 和 for 怎么区分
33. during、for、while 怎么区分
34. ago、before、in 怎么区分
35. how long、how often、how far、how soon 怎么区分

核心目标：
- 让读者不只会说一个“骨架句”
- 而是开始能补充时间、地点、频率、程度、距离、时长等信息

### 模块 5：动词系统扩展
这个模块解决“除了基础时态，还怎么用动词表达能力、义务、变化关系”。

36. 情态动词怎么区分
37. must 和 have to 怎么区分
38. can、could、be able to 怎么区分
39. 比较级和最高级怎么用
40. as...as 和 than 怎么区分
41. 被动语态怎么理解
42. 非谓语动词怎么入门

核心目标：
- 让读者从“会写基本动作句”走向“会表达能力、义务、比较、被动和结构变化”

### 模块 6：句子连接与复杂句
这个模块解决“如何从简单句走向更像真实英语的复杂表达”。

43. 连词怎么用
44. 宾语从句怎么入门
45. 定语从句怎么入门
46. 状语从句怎么入门

核心目标：
- 让读者学会连接句子
- 学会往句子里嵌入信息
- 建立复杂句的入门框架

### 模块 7：高频易混词与表达辨析
这个模块解决“不是不懂大规则，而是小词总说错”的问题。

47. say、speak、tell、talk 怎么区分
48. do 和 make 怎么区分
49. maybe、may be、perhaps 怎么区分
50. 英语里最常见的语法错误有哪些

如果后续继续扩写，这个模块还可以增加：
- to / too / two 怎么区分
- lend / borrow 怎么区分
- bring / take 怎么区分
- hear / listen / sound 怎么区分
- job / work 怎么区分

核心目标：
- 在系统主干讲完之后，集中解决高频表达误差
- 强化“会规则”和“会真正说对”之间的衔接

## 六、每篇文章固定模板

建议每篇继续保持统一结构：

1. **开头问题**
   - 从真实困惑、常见错误或最容易说反的地方切入
2. **本文重点**
   - 用一句话明确这篇只解决什么问题
3. **核心主线**
   - 先给最实用的判断框架，而不是先堆术语
4. **分项拆解**
   - 把规则按 2～4 个维度讲清楚
5. **最典型对比**
   - 用最容易混的句子做正面对照
6. **常见错误**
   - 尽量从中国学习者真实错误出发
7. **实用例句**
   - 让读者能直接套用
8. **小练习**
   - 3～5 题即可
9. **参考答案**
10. **一句话总结**
11. **结尾收束**
   - 自然收束，不再生硬引下一篇

---

## 七、内容风格建议

- 少讲抽象定义，多讲“怎么判断、怎么用”
- 一篇只解决一个点，但要明确它在整个系统里的位置
- 先讲主线，再讲例外
- 多做对比，不要只列规则
- 多写错句和修正句
- 口语化表达可以保留，但整体要有课程感和出版感
- 结尾以自然收束为主，不强推下一篇

标题方向继续保持：
- `英语语法里……怎么用？`
- `英语语法里……怎么区分？`
- `英语语法里……怎么入门？`
